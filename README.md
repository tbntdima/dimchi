# Usage
<!-- usage -->
```sh-session
$ npm install -g dimchi
$ dimchi COMMAND
running command...
$ dimchi (-v|--version|version)
dimchi/0.0.0 darwin-x64 node-v14.17.3
$ dimchi --help [COMMAND]
USAGE
  $ dimchi COMMAND
...
```

## `dimchi hello [FILE]`

describe the command here

```
USAGE
  $ dimchi hello [FILE]

OPTIONS
  -f, --force
  -h, --help       show CLI help
  -n, --name=name  name to print

EXAMPLE
  $ dimchi hello
  hello world from ./src/hello.ts!
```

_See code: [src/commands/hello.ts](https://github.com/tbntdima/dimchi/blob/v0.0.0/src/commands/hello.ts)_

## `dimchi help [COMMAND]`

display help for dimchi

```
USAGE
  $ dimchi help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```

_See code: [@oclif/plugin-help](https://github.com/oclif/plugin-help/blob/v3.2.2/src/commands/help.ts)_
